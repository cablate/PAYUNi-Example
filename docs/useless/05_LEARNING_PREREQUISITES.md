## 🆘 遇到困難怎麼辦？

### 學習過程中常見的困惑

1. **「我看不懂 Mermaid 圖」**
   - 沒關係，圖是輔助的，重點是文字說明
   - 可以搭配 AI 請它用白話文解釋流程

2. **「某個概念我還是不懂」**
   - 先標註起來，繼續往下讀
   - 回頭讀第二遍時通常會豁然開朗
   - 加入社群或講座 Q&A 提問

3. **「練習題我不知道答案對不對」**
   - 沒有標準答案！重點是你的思考過程
   - 可以在社群分享你的理解，與其他學員討論

4. **「我的環境跑不起來」**
   - 參考 `06_TROUBLESHOOTING.md`
   - 在社群提問時，附上完整錯誤訊息與環境資訊
