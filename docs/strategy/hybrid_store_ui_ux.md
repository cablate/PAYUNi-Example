# 混合式商品銷售 UI/UX 設計規劃 (Hybrid Store Design)

針對「訂閱制 (Subscription)」與「一次性商品 (One-time Purchase)」並存的場景，核心設計挑戰在於**降低使用者的認知負荷**，讓他們一眼就能區分「我是在買服務」還是「我是在買產品」。

以下是我的設計規劃：

## 1. 介面佈局 (Layout Strategy)

我建議採用 **「分頁切換 (Tabs)」** 或 **「區塊分層 (Sectioning)」** 的方式，視商品數量而定。

### 方案 A：頂部切換 (推薦用於商品較多時)
在「選擇方案」的標題下方，增加一個明顯的切換器 (Toggle Switch) 或分頁 (Tabs)。

*   **[ 💎 訂閱會員 ]**  vs  **[ 📦 精選商品 ]**
*   **預設狀態**：顯示「訂閱會員」，因為這是通常希望使用者優先選擇的長期價值方案。
*   **互動**：點擊切換後，下方的卡片列表會使用淡入淡出 (Fade in/out) 動畫切換，強調情境的轉換。

### 方案 B：上下分層 (推薦用於商品較少時)
如果一次性商品是作為「加購」或「補充」，可以放在訂閱方案下方。
*   **上層**：成為會員 (Subscribe) - 3 個卡片
*   **分隔線**：或是單次購買 (Or buy once)
*   **下層**：精選課程/禮包 - 網格排列

---

## 2. 卡片設計差異化 (Card Design Distinction)

利用視覺語彙來區分兩種購買類型：

### 🅰️ 訂閱制卡片 (Subscription Cards)
強調 **「週期性」** 與 **「服務持續性」**。
*   **價格顯示**：`$990 / 月` (強調 `/ 月` 或 `/ 年`)。
*   **行動呼籲 (CTA)**：使用「立即訂閱」、「開始免費試用」、「加入會員」。
*   **視覺特徵**：
    *   使用較為「柔和、科技感」的色調 (如藍色、紫色)。
    *   列出「權益 (Benefits)」而非單一規格。
    *   標籤：`熱門推薦`、`年繳省 20%`。
    *   **關鍵字**：解鎖 (Unlock)、無限存取 (Unlimited)、持續更新 (Updates)。

### 🅱️ 一次性商品卡片 (One-time Purchase Cards)
強調 **「擁有權」** 與 **「交付物」**。
*   **價格顯示**：`$3,500` (單一總價，可標示 `一次付費，終身擁有`)。
*   **行動呼籲 (CTA)**：使用「立即購買」、「加入購物車」、「購買課程」。
*   **視覺特徵**：
    *   使用較為「實體、溫暖」的色調 (如橙色、綠色) 或強調商品圖片。
    *   如果有實體商品，必須顯示 **商品縮圖 (Thumbnail)**。
    *   **關鍵字**：終身觀看 (Lifetime Access)、實體寄送 (Shipping)、立即下載 (Download)。

## 3. 資料結構調整 (Data Structure)

為了支援這種顯示，我們需要在 `src/data/products.js` 中增加 `type` 欄位：

```javascript
// 範例資料結構
{
  id: "plan_pro",
  type: "subscription", // 關鍵欄位
  name: "專業方案",
  price: 1990,
  period: "/月", // 訂閱制專有
  features: ["..."]
},
{
  id: "course_react",
  type: "one_time", // 關鍵欄位
  category: "course", // 子分類：course, physical, digital
  name: "React 實戰課程",
  price: 3500,
  period: null, // 一次性沒有週期
  image: "/images/react-course.jpg", // 一次性商品通常需要圖片
  features: ["終身觀看", "附贈原始碼"]
}
```

## 4. 結帳流程微調 (Checkout Flow Nuances)

*   **訂閱制**：
    *   必須強調「自動續扣」的條款。
    *   結帳時顯示「下一次扣款日」。
*   **一次性商品**：
    *   如果是實體商品，結帳流程 **必須增加「收件地址」欄位**。
    *   如果是虛擬商品，強調「購買後至 Email 收取」。

## 5. 具體畫面想像 (Visual Imagination)

想像目前的畫面：
1.  **Step 2 標題**：由 "Select Plan" 改為 "Choose Your Path" (選擇您的方案)。
2.  **控制項**：一個膠囊狀的切換器 (Capsule Toggle)。
    *   左邊：**訂閱服務** (選中時背景亮藍色)
    *   右邊：**單次購買** (選中時背景亮橙色)
3.  **內容區**：
    *   切換到「訂閱」時，顯示我們剛做好的 3 個直式方案卡片。
    *   切換到「單次」時，顯示橫式的商品列表，左邊是圖片，右邊是標題與價格。

這樣的設計既能保留 SaaS 的專業感，又能靈活擴充販售其他類型的商品。
